# Insurance Policy Management System

Overview
This project is a simple insurance management system built in Python using Object-Oriented Programming (OOP).

Project Structure
- policyholder.py → Handles policyholder operations
- product.py → Handles insurance products
- payment.py → Handles payments
- main.py → Demonstrates system functionality

Features
- Register, suspend, and reactivate policyholders
- Create, update, and suspend products
- Process payments, apply penalties, and send reminders
